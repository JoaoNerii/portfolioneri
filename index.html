BEM-VINDO AO MEU PORTFÓLIO PESSOAL

Aqui você vai encontrar os projetos e conquistas que refletem minha jornada no mundo da programação e tecnologia. Cada trabalho representa uma nova etapa de aprendizado e superação, com foco em desenvolvimento de software e soluções criativas. Explore, descubra e acompanhe essa caminhada em constante evolução.

MUITO PRAZER, ME CHAMO JOÃO NERI

Minha paixão por tecnologia começou cedo, inspirado pelas primeiras vezes que vi meu pai utilizando o computador. A curiosidade foi crescendo e me levou a descobrir o universo da programação, onde encontrei meu verdadeiro interesse. Desde então, venho explorando linguagens como Java e mergulhei de vez no curso de Sistemas de Informação. Aqui compartilho minha trajetória, desafios e tudo o que estou aprendendo nessa jornada.

MINHAS ESPECIALIDADES.

Python

Minha jornada com Python começou quando me deparei com tutoriais e projetos simples na internet. O que mais me atraiu foi a simplicidade da linguagem e sua versatilidade. Python me ensinou a base do raciocínio lógico e como transformar ideias em código funcional. Desde automação de tarefas até manipulação de dados, descobri como essa linguagem é poderosa para diversos cenários. Embora ainda tenha muito a aprender, cada projeto com Python reforça minha paixão pela programação.

Java

Java foi a primeira linguagem que estudei de maneira mais aprofundada, iniciando com um curso online no começo deste ano. Logo percebi a importância dessa linguagem no desenvolvimento de softwares robustos e escaláveis. Java me desafiou a entender conceitos fundamentais de orientação a objetos, além de como construir soluções eficientes para problemas complexos. Com sua ampla aplicação no mercado, tenho buscado me aprofundar cada vez mais, com foco em desenvolvimento de sistemas.

HTML

HTML foi meu primeiro contato com o mundo do desenvolvimento web. Entender como as páginas que vemos na internet são estruturadas e construídas me fez admirar ainda mais o universo digital. Aprender HTML foi essencial para compreender como funciona a base de qualquer site, e aos poucos, fui evoluindo minhas habilidades de marcação, criando páginas estáticas e entendendo a importância da semântica. HTML é o alicerce para qualquer front-end, e sigo aprimorando minhas habilidades para projetos futuros.
