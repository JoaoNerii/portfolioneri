Começando pelo meu último projeto, a LocaSmart

Desenvolvido para a matéria de Projetos da faculdade, a LocaSmart é um site para reserva de carros on-line. O Projeto ainda está em fase de desenvolvimento, nele iremos fazer um site usando Python, HTML e Css para atender todos requisitos de uma locadora online.

PROCESSO DE CRIAÇÃO LOCASMART.
